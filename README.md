# Fourier_audio
*Trataramento de áudio através da análise de Fourier*

O código em MATLAB apresenta primeiramente a análise de Fourier de uma onda quadrada.
Depois, é feita a filtragem das frequências de 2 áudios através de filtros passa alta e baixa

Desenvolvido em Aula na FEI junto ao Prof. Vanderlei
